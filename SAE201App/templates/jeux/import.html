Importation de la liste des jeux

Pour importer une liste de jeux, vous devez utiliser un fichier au format JSON.

Ce fichier JSON doit être de la sorte :

{
    "$schema": "https://raw.githubusercontent.com/JM-UHA/SAE203/main/schema/import.schema.json",
    "jeux": [
        {
            "nom": "Nom du jeu",
            "editeur": "Nom de l'éditeur",
            "annee": "Année de sortie",
            "categorie": "Catégorie du jeu",
            "description": "Description du jeu",
        },
        {
            "nom": "Nom du jeu",
            "editeur": "Nom de l'éditeur",
            "annee": "Année de sortie",
            "categorie": "Catégorie du jeu",
            "description": "Description du jeu",
        },
        {
            ...
        }
    ]
}

La clé $schema permet d'ajouter de l'autocomplétion lors de l'écriture du JSON avec les éditeurs compatibles. (Notamment VSCode)
